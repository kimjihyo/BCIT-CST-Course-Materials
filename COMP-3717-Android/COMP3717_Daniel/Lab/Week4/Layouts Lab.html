Download & extract TipCalculator.zip in a working directory. Load the Tip Calculator app into Android Studio. When you run the application you will notice there are five buttons on the main activity, namely:

  • Constraint Layout
  • Linear Layout
  • Relative Layout
  • Grid Layout
  • Table Layout

The only button that is somewhat working is "Constraint Layout".

  • Click on the "Constraint Layout" button and view the ConstraintActivity UI
  • Add any amount to Bill Amount then hit Enter. A 15% tip is automatically added to Bill Amount. 
  • Click on - or + to decrease or increase tip percentage respectively

PROBLEM: If you rotate the phone, all the values are lost.

Your mission

1) Fix problem with rotation so that information is not lost.

2) Get all the other buttons on activity_main.xml working by implementing the same UI as ConstraintActivity using Linear, Relative, Grid and Table layouts. For all additional layouts, it is necessary for the tip calculator to behave properly and that the UI looks very similar to the ConstraintActivity. Also, make sure that UI adjusts itself appropriately in portrait and landscape modes.
