Objective

The objective of this lab is to ensure that the Android Development Environment is properly setup and works on your laptop. Therefore, you will show your instructor a working app running in an emulator on your laptop and not on a physical device.

The task an hand

Create an android app. Give it an EditText control (referred to as a Plain Text control in Android Studio) and a Button with text "Reverse Text". When the user enters text in the EditText control and clicks on the "Reverse Text" button, text is the the EditText control is reversed and placed in a TextView control below the button. This is what the UI should look like:

For example, when 'Canada' is entered, it gets reversed to 'adanaC'. Display error message 'You must enter text.' in red if the user enters nothing then clicks on the button.

The button event handler method name must be coded in the XML resource file. The event handler code is in java.

This lab is to be done in class. Make sure to show the working app to your lab instructor running in an emulator before the end of the lab.

Best practices

  • Remember to put all your static strings in the strings.xml file.
  • Give your controls meaningful names and avoid names like Button1, TextView1, etc..
